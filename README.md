# Open discussions

Scientific progress starts with good questions.

This repository collects open scientific questions related to Myrionyx projects.

Examples include:

* Can biological mirror states be formally defined?
* Which mathematical framework best represents biological mirroring?
* What types of measurements could support conceptual EMP-inspired diagnostics?
* How can interdisciplinary collaboration improve theoretical development?

Researchers from all disciplines are invited to propose new questions and discuss existing ones.
